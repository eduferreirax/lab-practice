# Lab Practice

This repository contains my **hands-on penetration testing practice** using training machines from platforms such as **HackingClub** and other lab environments.

The goal of this repository is to document the **methodology, tools, and techniques** used during the exploitation of vulnerable machines.

This is part of my journey toward becoming a **professional penetration tester** and preparing for certifications such as **CPTS (Certified Penetration Testing Specialist)**.

---

# Objectives

- Practice real-world penetration testing scenarios
- Improve enumeration and exploitation skills
- Document techniques and workflows
- Build a personal knowledge base

---

# Methodology

Each machine typically follows this workflow:

1. **Reconnaissance**
   - Network scanning with Nmap
   - Service identification

2. **Enumeration**
   - Web enumeration
   - Directory discovery
   - Service analysis

3. **Exploitation**
   - Identifying vulnerabilities
   - Gaining initial access

4. **Privilege Escalation**
   - Linux or Windows privilege escalation
   - Misconfigurations
   - SUID / weak permissions / credential leaks

5. **Post Exploitation**
   - System enumeration
   - Flag retrieval
   - Documentation
